Getting Started

Hello, and thank you for purchasing my theme, Trendsetter. I hope you enjoy using it!

To start, extract the package and upload its contents to your server. Index.html is the template file and should be functional without further setup.

The contact form will need to modified so that it sends emails to the correct address, more details about that here.

Ensure that PHP is supported by your server, or else the contact form script won't work.

The dummy text included in the template is hard-coded into the page. Change it to your own content.

The favicon for the site is favicon.png. Change this image to change your favicon.


Pages

This theme relies on an index.html page and a series of pages for your portfolio items (one dedicated page for each of your portfolio items). In the demo package, they are the "project_*.html" files.

The index.html page will contain the bulk of your info. All your text and images, excluding your portfolio items, are hard coded into the index page.

The template is divided into the home, portfolio, about, and contact sections. Change the text and images in index.html to edit the content. The start and end of each section is commented in the HTML. To add or remove sections, copy and paste or delete the sections and update the navigation bar.


Colors

The theme uses two primary colors that can be adjusted in css/main.css. The elements that use colors are listed at the top of the stylesheet, along with color codes of the two default colors. Replace the color code in each of those elements using a simple find and replace with your new two colors.

Alternatively, include one of the available color stylesheets in css/color/COLOR.css into the head of index.html after the main.css reference. An example blue css stylesheet is commented out in the header.


Navigation Bar

The navigation bar uses Bootstrap Scrollspy. The links in the navigation bar link to the id of the appropriate section. When adding a section, give the first div in the section an id and use that as the href target in the navigation link.


Background Images

Each content section has a large background image. To change these images, find the class masthead-image parallax at the bottom of the section and change the src to your own image. Alternatively, create your own image and overwrite the existing image by using the same filename.

The rough dimensions of the background images are 800 x 518. It is fine to vary from this as long as your image dimension ratio is in the same ballpark.

Each background image has a parallax effect. If you can't see it, slowly scroll with the scrollbar to see the effect.


Home Section

Set your name and title by replacing the default generic name in the h1 class="intro-text" tag and h2 class="intro-secondary" tag.

You can change the font size of the text easily at the top of css/main.css

Link to your social media pages with the social link icons. The social icons, and the rest of the icons in this template, are from Font Awesome. See their site for a list of all available icons (newer releases of icons may not be included in this template, but you can download and replace the current version).

Icons are created using the <i> class. To learn more about how to use font, visit Font Awesome.

To change the social media icon, change class="fa fa-twitter" to class="fa fa-linkedin" for example.


Portfolio Section

Each item in the portfolio section has its own page which is loaded through AJAX when you click on one of the portfolio links.

The portfolio_*.html images in the template's root folder show examples of different types of portfolio items available (image, image gallery, video).


Filter

This template uses the JQuery Isotope plugin to create the sorting effect for the portfolio links. To read how it works, click here.

To set the categories, find the option-isotope ul list in the portoflio section. Each list item has a data-option-value which specifies which category class to show, and the HTML text inside the list item's anchor link is what is actually displayed on the page as the category name (such as Photo).

Every sortable element in the template uses the class element to be designated as one of the sortable items. To sort items into categories, they have to share a common class (like category1. You can keep the category classes as category1, category2, and so on, because they are not actually displayed. All the items in category1 will be shown when you click on the list link with data-option-value=".category-1". Note that the data-option-value contains a . since it is targeting a class.


Thumbnail

Each portfolio item shows a thumbnail, title, category, and quick description. Simply replace the sample data with your own text and image source.

Clicking the thumbnail loads the actual content into the page in a popup. Simply specify the page in the anchor link's href like a normal link and the JavaScript will handle the rest.


Portfolio item page

Each portfolio item should have its own page that you link to from the thumbnail. These pages are the project_*.html files in the sample.

The page that the portfolio item link loads doesn't have any style on it's own until it gets loaded into the main page. Follow the structure of the existing sample portoflio items and replace their text and images with your own.

Galleries are created with the standard Bootstrap Carousel.

To embed a video from Youtube or Vimeo, grab the embed code from your video and remove all the excess code that you get from it (such as the anchor link to the video,

Whether your main content is an image, gallery, or video, give it the class portfolio-image-full like in the samples.


About Section

Replace the content in the About section with your own text. The progress skill bars can be changed by adjusting the width % in the style of the element.


Contact Section

The google maps used in the contact form uses a simple embed from the Google Maps service. Be sure to strip the default embed code that you get from Google Maps of the width and height settings.

The contact form is PHP based so ensure that you use it on a server that supports PHP with mail.

The success message for contact form submission can also be changed here.
